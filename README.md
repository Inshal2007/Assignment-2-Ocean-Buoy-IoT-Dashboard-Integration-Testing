# Project II – Assignment #2  
## Ocean Buoy IoT Dashboard & Integration Testing

## Overview

This project defines the user interface requirements and integration testing strategy for an IoT-based ocean buoy monitoring system supporting a salmon aquafarm.

The buoy collects environmental and positional data and transmits it to a centralized system. The dashboard visualizes this data and generates alerts when predefined safety or anomaly thresholds are violated.

## Data Collected by the Buoy

- Maximum Wave Height
- Minimum Wave Height
- Latitude
- Longitude
- Water Temperature
- Air Temperature
- Oxygen Saturation (%)

## System Objectives

- Monitor environmental conditions affecting salmon health
- Detect buoy drift (geofence breach)
- Detect abnormal spikes in sensor readings
- Provide real-time alerts for critical thresholds
- Maintain data integrity and freshness visibility
